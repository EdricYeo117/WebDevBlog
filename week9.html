Bootstrap grid system:

  • CSS Flexbox and Grid System

    • Helps tremendously to create flexible and responsive layouts, based on a 12-column layout, facilitating complex designs

  • Pre-designed HTML, CSS and Javascript components, such as nav bar, buttons, forms and modals

    • Types of charts:

  • Line Charts

    • Plotting data points on a line, used to show trend data or comparison of two data sets

  • Radar Charts

    • Shows multiple data points and variation between them, often useful for comparing the points of two or more different data sets

  • Pie & Donut Charts

    • Each segment shows proportional value of each piece of data, excellent at showing relational proportions between data
